Here's a brief run down of the apps functionality.

  • Move to the next sibling node with "Control left" and "Control right"
  • Move to the first child node with "Control Down"
  • Move to the parent node with "Control up"
  • Control node depth visibility by typing "/" to get to the input and then a number to determine how many levels of node you wish to view (max capability is four)
